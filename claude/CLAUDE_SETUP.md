# Claude Code Skills

Claude Code ships with built-in skills (e.g. `/review`, `/init`, `/schedule`). The following 6 are installed separately from the official marketplace:

```sh
claude plugins install xlsx
claude plugins install pdf
claude plugins install pptx
claude plugins install docx
claude plugins install mcp-builder
claude plugins install skill-creator
```

Source: [`anthropics/claude-plugins-official`](https://github.com/anthropics/claude-plugins-official)
