# HTML Themes

Prompt templates for generating themed HTML pages from AI daily news.

## Apple Style (苹果风)

Clean, minimalist design inspired by Apple's aesthetic. Best for technical content.

**Design**:
- Pure black background (#000000)
- Subtle blue glow from bottom-right (#0A1929 → #1A3A52)
- Generous white space, content density ≤ 40%
- SF Pro Display for headings, SF Pro Text for body (fall back to system sans-serif)
- Smooth animations and hover effects

**Structure**:
- Header: Logo icon + Date badge
- Main: Summary card + Category sections
- Footer: Keywords + Copyright

**Colors**:
- Title: #FFFFFF
- Body text: #E3F2FD
- Accent: #42A5F5
- Secondary: #B0BEC5
- Card background: rgba(255, 255, 255, 0.05)
- Card border: rgba(66, 165, 245, 0.2)

## Deep Ocean (深海蓝)

Blog-style layout with deep blue tones. Best for product launch summaries.

**Design**:
- Dark navy gradient background (#0D1B2A → #1B2838)
- Soft cyan accents
- Card-based layout with subtle shadows
- System sans-serif fonts

**Structure**:
- Header: Title bar with date
- Main: Grid of category cards
- Footer: Source attribution

**Colors**:
- Title: #E0F7FA
- Body text: #B2EBF2
- Accent: #00BCD4
- Secondary: #78909C
- Card background: rgba(13, 27, 42, 0.8)
- Card border: rgba(0, 188, 212, 0.3)

## Autumn Sun (秋日暖阳)

Warm, inviting design. Best for community and ecosystem news.

**Design**:
- Warm cream background (#FFF8E1 → #FFECB3)
- Orange and amber accents
- Rounded corners, friendly typography
- System sans-serif fonts

**Structure**:
- Header: Playful title + Date
- Main: Alternating left/right content blocks
- Footer: Keywords cloud + Source

**Colors**:
- Title: #E65100
- Body text: #4E342E
- Accent: #FF9800
- Secondary: #8D6E63
- Card background: rgba(255, 255, 255, 0.7)
- Card border: rgba(255, 152, 0, 0.3)
